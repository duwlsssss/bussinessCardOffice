# BusinessCardFactory
졸업 전시 명함사무실 

To setup a dev environment:

```bash
# Clone the repository
git clone https://github.com/duwlsssss/bussinessCardOffice.git

# download models
# put these at public/models 
# dancer.glb
https://drive.google.com/file/d/1hMXJCU9lrBweVnczSCgPnM2eRdxb3cyI/view?usp=sharing
# map.glb
https://drive.google.com/file/d/1vN6RaUcd5Xfk0Zubkni31CAuKfDEtvuU/view?usp=sharing
# monitor.glb
https://drive.google.com/file/d/1w5FRVMhxRjWvuvJL5wEozLPkEsB2vdN4/view?usp=sharing
# office.glb
https://drive.google.com/file/d/12yxOrs84EOxEV7OSHCDl4UdtuKU-L65a/view?usp=sharing
# standing_55.glb
https://drive.google.com/file/d/1l5kD3p2dgdlT9kMS9PY5pCGk3SVmWAe8/view?usp=sharing

# Install dependencies 
npm i

# Run the local dev server
npm run dev
```

To serve a production build:

```bash
# Install dependencies if not already done - 'npi i'

# Build for production
npm run build

```
